## FIFA Player Database Management System
A menu-driven Python application for managing, analyzing, and visualizing FIFA player data using MySQL and Pandas.
This project focuses on database operations, data analysis, and secure coding practices.
## Project Overview
The FIFA Player Database Management System allows users to:
- View player records stored in a MySQL database
- Insert, update, and delete player data
- Perform basic data analysis using Pandas
- Visualize player data using graphs
- Export the current database state to a CSV file
